Google Text To Speech Rest API

 

Api Ana sayfası : https://cloud.google.com/text-to-speech

 

Etkinleştirilecek projeyi seçin veya oluşturup etkinleştirin:

https://console.cloud.google.com/flows/enableapi?apiid=texttospeech.googleapis.com

 

Api URL:https://texttospeech.googleapis.com/v1beta1/text:synthesize?key=x

Not : x değerini kendi api anahtarınız ile değiştirin!

key değeri Text to speech api anahtarıdır Google cloud üzerinden API'ler ve Hizmetler>Kimlik bilgileri sayfasından Kimlik bilgisi oluştur a tıklayarak API anahtarı oluşturularak elde edilir aktif olması zaman alabilir yeşil tik işareti belirince kullanılabilir

synthesize konuşmayı senkronize olarak sentezler: tüm metin girişleri işlendikten sonra sonuçları alırsınız.

Talep gövdesi (json)

{ 
  "input" : { 
    object ( SynthesisInput) 
  } , 
  "voice" : { 
    object ( VoiceSelectionParams) 
  } , 
  "audioConfig" : { 
    object ( AudioConfig) 
  } 
}

 

Alanlar

input: Gereklidir. Sentezleyici, giriş olarak düz metin veya SSML gerektirir.

voice: Gereklidir. Sentezleyici sesini belirler.

https://texttospeech.googleapis.com/v1beta1/voices?key=x adresinden liste mevcuttur

audioConfig: Gereklidir. Sentezlenen sesin konfigürasyonu.

SynthesisInput

Sentezlenecek metin girişini içerir. Ya text veya ssml sağlanmalıdır. Giriş boyutu 5000 karakterle sınırlıdır.

{ 
  // Aşağıdakilerden yalnızca biri olabilir: 
  "text" : string , 
  "ssml" : string 
}

VoiceSelectionParams

Bir sentez isteği için hangi sesin kullanılacağını belirler.

 

{ 
  "languageCode" : string , 
  "name" : string , 
  "ssmlGender" : enum ( SsmlVoiceGender)
}

 

 

languageCode: Gereklidir. BCP-47 dil etiketi olarak ifade edilen sesin dili (ve ayrıca potansiyel olarak bölgesi) , örneğin "en-US". 

 

name: Sesin adı. Ayarlanmazsa, hizmet, languageCode ve cinsiyet gibi diğer parametrelere dayalı bir ses seçer.

 

 

ssmlGender: Sesin tercih edilen cinsiyeti. Ayarlanmazsa, hizmet, languageCode ve name gibi diğer parametrelere dayalı bir ses seçer. Bunun sadece bir tercih olduğunu, gereklilik olmadığını unutmayın; uygun cinsiyete sahip bir ses mevcut değilse, sentezleyicinin sesi, isteği başarısız olmak yerine farklı bir cinsiyetle değiştirmesi gerekir.

AudioConfig

{
  "audioEncoding": enum (AudioEncoding),
  "speakingRate": number,
  "pitch": number,
  "volumeGainDb": number,
  "sampleRateHertz": number,
  "effectsProfileId": [
    string
  ]
}

 

audioEncoding: Gereklidir. Ses bayt akışının biçimi.

 

LINEAR16

Sıkıştırılmamış 16 bit imzalı küçük endian örnekleri (Doğrusal PCM). LINEAR16 olarak döndürülen ses içeriği de bir WAV başlığı içerir.

MP3

32kbps hızında MP3 ses.

OGG_OPUS

Bir ogg kabına sarılmış Opus kodlu ses. Sonuç, Android'de ve tarayıcılarda (en azından Chrome ve Firefox) yerel olarak oynatılabilen bir dosya olacaktır. Yaklaşık olarak aynı bit hızını kullanırken kodlamanın kalitesi MP3'ten oldukça yüksektir.

 

 

speakingRate: İsteğe bağlı. Yalnızca giriş. [0.25, 4.0] aralığında konuşma hızı / hızı. 1.0, belirli bir ses tarafından desteklenen normal yerel hızdır. 2.0 iki kat hızlı ve 0.5 yarı yarıya. Ayarsız (0,0) ise, varsayılan olarak yerel 1,0 hızına ayarlanır. Diğer tüm değerler <0,25 veya > 4,0 ise hata döndürür.

 

pitch: İsteğe bağlı. Yalnızca giriş. Konuşma aralığı, [-20.0, 20.0] aralığında. Şekil 20, orijinal aralıktan 20 yarıton artırılması anlamına gelir. -20, orijinal aralıktan 20 yarıtonun azaltılması anlamına gelir.

 

sampleRateHertz: İsteğe bağlı. Bu ses için sentez örnekleme hızı (hertz cinsinden). Bu, belirtildiğinde, bu sesin doğal örnekleme hızından farklıysa, belirtilen örnekleme hızı desteklenmedikçe, sentezleyici istenen örnekleme hızına (daha kötü ses kalitesine neden olabilir) dönüşerek bu isteği yerine getirecektir.

Response body

Başarılı olursa, yanıt gövdesi aşağıdaki yapıya sahip veriler içerir:

 

{
  "voices": [
    {
      object (Voice)
    }
  ]
}

 

voice: Binary ses dosyasının base64 halidir decode edilerek kullanılabilir.

Örnek

Request URL : https://texttospeech.googleapis.com/v1beta1/text:synthesize?key=x

 
Request body
 
{
  "audioConfig": {
    "audioEncoding": "LINEAR16",
    "pitch": 0,
    "speakingRate": 1
  },
  "input": {
    "text": "Deneme 1 2 3"
  },
  "voice": {
    "languageCode": "tr-TR",
    "name": "tr-TR-Standard-A"
  }
}
